espeak PC版本编译配置

介绍

eSpeak is a compact open source software speech synthesizer for English and other languages, for Linux and Windows. http://espeak.sourceforge.net eSpeak uses a "formant synthesis" method. This allows many languages to be provided in a small size. The speech is clear, and can be used at high speeds, but is not as natural or smooth as larger synthesizers which are based on human speech recordings. espeakedit is a GUI program used to prepare and compile phoneme data. It is now available for download. Documentation is currently sparse, but if you want to use it to add or improve language support, let me know. 官方介绍

espeak 音频重采样

由于espeak默认发音音元都是22050HZ_S16采样率的，而设备端默认是8000_U8格式，需要做重采样

  • 参考
    • https://cloud.tencent.com/developer/article/1094594
    • https://github.com/mackron/dr_libs/blob/master

  • 修改代码 参考这个博主的代码后，可以把22050HZ转为8000HZ，再拷贝修改resampledata函数中的int16部分的到resampledata8bit函数

# espeak audio resample
